JSX 語法是一種語法糖，近似於 JavaScript 的語法擴充，放棄模板語言的形式，採用更語意化可讀性更強的標籤。並且使用 {} 內嵌任何 JavaScript 表達式。

組件書寫特性

自定義 component 首字需大寫，這樣 JSX 才能識別為自定義還是原始 HTML 標籤。

Props 屬性

組件近似於函式，接受特定的輸入 (props)，產出特定的輸出 (React elements)。
